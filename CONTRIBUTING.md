# Contributing to git-branch-janitor

Contributions are welcome. Here's how to get set up.

## Setup

```bash
git clone https://github.com/jishanahmed-shaikh/git-branch-janitor.git
cd git-branch-janitor
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -e "." pytest
```

## Running Tests

```bash
pytest
```

## Guidelines

- One feature or fix per PR
- Add tests for any new logic
- Follow PEP 8
- Update README if behavior changes

## Issues

Open an issue at [GitHub Issues](https://github.com/jishanahmed-shaikh/git-branch-janitor/issues).
